Task Management System

Overview

Task Management System (TMS) is a browser extension for managing your Microsoft Tasks. TMS stores your tasks locally on your computer, so that you can view them without an Internet connection.

System Requirements

You will need the following to run Task Management System:

  • An Internet connection (not all the time).
  • A modern web browser like Firefox, Chrome or Edge.
  • A user account with one of Microsoft's online services (Hotmail, Office 365, Outlook).

Task Management System (TMS) is not a replacement for the actual Microsoft Tasks app, because it does not support all the features that the Microsoft Tasks app has.

Operation

The first time you launch the app you will need to click "Login" which will take you to the Microsoft Authorization page where Microsoft will ask you whether you allow this app to access your Microsoft Tasks data.

Task Management System (TMS) will not update your tasks or folders unless you click on the refresh icons at the top.

Click on the folder on the left to view the tasks in the center.

Click on the task in the middle to edit the task on the right.

User Interface

Install

Time Management System (TMS) can be installed from these sources:

  • Mozilla Firefox extensions
  • Google Chrome extensions
  • Microsoft Edge extensions

Time Savers

  • To quickly click on the Save button after choosing a date or time, simply double-click on the date or time.
  • The date and time picker can be navigated using the keyboard by using the 4 arrow keys. To get to the date and time picker while you are typing a new task, press the tab key multiple times until the keyboard focus reaches the date and time picker.
